# Security Policy

## Supported Versions

Only these versions are supported:

| Version | Supported          |
| ------- | ------------------ |
| Stable / LTS   | ✅ |
| Bug Fix   | ✅                |
| Experiment   | :x: |
| Non-Oficial Releases   | :x:                |
